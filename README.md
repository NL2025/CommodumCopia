## Introductie
Dit is de frontend voor de Commodum Copia supermarkt website. Het is gebouwd met Next.js 14 en maakt gebruik van de App Router voor routing en Tailwind CSS voor styling. De website bevat een navigatiestructuur, een datastructuur voor producten en een kleurschema dat past bij de huisstijl van Commodum Copia.
